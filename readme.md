# Schedule scrapper

## Description

Scrap all students groups schedules

### [Schedule example](https://www.voenmeh.ru/trainee/timetable-stud)

## Stack

* ### cheerio
* ### selenium-webdriver
* ### sqlite3

### Install

```
yarn
```

### Development

```
yarn dev
```

### Build

```
yarn build
```
